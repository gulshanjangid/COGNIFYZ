C++ Variables and Constants

C++ Variables

Variables are containers that store information that can be manipulated and referenced later by the programmer within the code.

or

Variables serve as containers for storing data values.

In the realm of C++, a variety of variable types exists. Here are some of them:

  • An integer variable, defined with the keyword "int," holds whole numbers without decimals, such as 63 or -1.

  • A floating-point variable, defined with the keyword "float," stores numbers with decimals, like 79.97 or -13.26.

  • A character variable, specified with the keyword "char," stores single characters like 'A' or 'z'. Char values are enclosed within single quotes.

  • A boolean variable, declared with the keyword "bool," holds a singular value: 0 for false and 1 for true.

How to Declare Variables?


// Declaring a single variable
type variable_name;
// Declaring multiple variables:
type variable1_name, variable2_name, variable3_name;


Guidelines for Variable Declaration:

  • The variable name can consist of letters, digits, and underscores.

  • Variable names are case-sensitive; for instance, "Arr" and "arr" represent distinct variables.

  • Variable names should not include any whitespace or special characters (e.g., #, $, %, *).

  • All variable names must commence with a letter of the alphabet or an underscore (_).

  • Usage of C++ keywords (such as float, double, class) as variable names is not allowed.

Valid Variable Names:

int x; // Can contain letters
int _yz; // Can include underscores
int z40; // Can have a mix of letters and numbers


Invalid Variable Names:

int 89; // Should not begin with a number
int a b; // Should not contain any whitespace
int double; // Cannot use a C++ keyword as a variable name


Distinguishing Between Definition and Declaration

In programming, variable declaration signifies the initial introduction of a variable before its inaugural use. On the other hand, a variable definition is the segment where the variable is allocated a memory location and assigned a specific value.



C++ Constants

In C programming, a constant is a variable that, true to its name, remains unalterable once declared in the program. The values of constant variables cannot be modified once they are defined.


const int LIGHT_SPEED = 299792458;
// Attempting to modify LIGHT_SPEED will result in an error.
// Example:
// LIGHT_SPEED = 2500;
// Error! LIGHT_SPEED is a constant.


What is the Difference between Constant and Variables?

Constant:-

  • A constant does not change its value over time.

  • Constants are usually written in numbers.

  • Constants usually represent the known values in an equation, expression or in line of programming.

  • Constants have fixed face values.

Variables:-

  • A variable, on the other hand, changes its value dependent on the equation.

  • Variables are specially written in letters or symbols.

  • Variables, on the other hand, represent the unknown values.

  • Variables do not have fixed face values.
