C++ Pointers

A pointer is a data type that stores the address of another data type. Essentially, a pointer is a variable that can point to any other variable and can be of types like int, char, array, function, or even another pointer. Pointers in C++ are declared using the ‘*’ (asterisk) operator.

The ‘&’ (ampersand) operator is known as the ‘address of’ operator, and the ‘*’ (asterisk) operator is referred to as the ‘value at’ or dereference operator.

Applications of a Pointer

  1. Dynamic Memory Management:

    • Pointers are utilized for dynamic allocation and deallocation of memory, allowing for flexible memory usage during program execution.

  2. Container Addressing:

    • Pointers are employed to reference various containers like arrays or structs. They are also useful for passing addresses of containers to functions.

  3. Returning Multiple Values from a Function:

    • Pointers enable the return of multiple values from a function by passing the addresses of variables.

  4. Memory Efficiency:

    • Rather than passing copies of containers to functions, pointers can be passed, leading to reduced memory usage in the program.

  5. Code Optimization:

    • The use of pointers often results in more concise code and improved program performance.

Operations on Pointers

1.Address of Operator (&):-

The & operator is also recognized as the Referencing Operator, and it functions as a unary operator. The variable name used in conjunction with the Address of operator must correspond to the name of an already defined variable.

Employing the & operator with a variable provides the memory address of that variable.


Indirection Operator:-

* is also known as the Dereferencing Operator. It is a unary operator. It takes an address as its argument and returns the content/container whose address is its argument.


Pointer to Pointer:-

The concept of a Pointer to Pointer involves storing the address of one pointer in another pointer. This is often termed as multiple indirections due to the operator’s name. In this scenario, the first pointer contains the address of the second pointer, which, in turn, points to the address where the actual variable stores its value.


Arrays and Pointers:-

Storing the address of an array into a pointer differs from storing the address of a variable into a pointer. The name of an array represents the address of its first index. Therefore, using the (ampersand)& operator with the array name to assign the address to a pointer is incorrect. Instead, the array name itself is used for this purpose.

int marks[] = {99, 100, 38};
int *p = marks;
cout << "The value of marks[0] is " << *p << endl;

Output:

The value of marks[0] is 99
