C++ Introduction

What is C++?

  • Bjarne Stroustrup conceived C++ as a natural extension of the venerable C programming language.

  • Despite its origin in the 1980s, C++ has sustained its popularity as a programming language over the ensuing decades.

  • Functioning as a cross-platform language, C++ empowers developers to craft high-performance applications and robust software systems.

  • With its intimate connection to hardware, C++ facilitates direct programmer instructions to the system, eschewing intermediaries. This hands-on approach grants programmers an elevated degree of control over system resources and memory.

Features of C++?

  • C++ stands as one of the most widely embraced programming languages globally.
    In the contemporary landscape of operating systems, graphical user interfaces (GUIs), and embedded systems, C++ holds a pervasive presence.
    Renowned for its object-oriented approach, C++ furnishes programs with a discernible structure, fostering code reusability and diminishing development expenses.
    This programming language empowers the creation of applications and resource-intensive games capable of running seamlessly across diverse platforms.
    Given its proximity to other languages like C# and Java, transitioning to or from C++ is a seamless process for programmers, and its learnability remains notably straightforward.


Distinguishing Between C and C++

  1. Programming Paradigm:-

    • C: Procedural programming language.

    • C++: Object-oriented programming (OOP) language with procedural programming capabilities.

  2. Approach to Programming:-

    • C: Emphasizes functions and procedures.

    • C++: Focuses on objects and classes, allowing for both procedural and object-oriented programming.

  3. Code Structure:-

    • C: Lacks support for classes and objects, resulting in a flatter structure.

    • C++: Incorporates classes and objects, promoting a more hierarchical and modular code structure.

  4. Memory Management:-

    • C: Relies on functions like malloc() and free() for dynamic memory allocation and deallocation.

    • C++: Introduces the new and delete operators for memory management, along with features like constructors and destructors.

  5. Function Overloading:-

    • C: Does not support function overloading.

    • C++: Allows multiple functions with the same name but different parameters through function overloading.

  6. Namespace Support:-

    • C: Does not have a namespace feature.

    • C++: Supports namespaces to avoid naming conflicts in larger projects.

  7. Exception Handling-

    • C: Lacks built-in support for exception handling.

    • C++: Provides try, throw, and catch mechanisms for effective exception handling.

  8. Standard Template Library (STL):-

    • C: Does not have a standard template library.

    • C++: Includes the STL, offering a collection of template classes and functions for various data structures and algorithms.
